# User Manual: Azuralis Healthcare Platform

## Introduction

The Azuralis Healthcare Platform is a comprehensive medical record management system designed for healthcare facilities. This manual provides guidance for different user roles: Clinical Staff (Doctors/Nurses), Patients, and Guardians.

## Getting Started

### System Requirements

- **Browser**: Chrome, Firefox, Safari, or Edge (latest version)
- **Device**: Desktop, tablet, or mobile
- **Connection**: Stable internet connection
- **JavaScript**: Must be enabled

### First Login

1. Navigate to the application URL
2. Enter your email address
3. Enter your password
4. Click "Iniciar Sesión" (Sign In)
5. Complete any required profile information

### Changing Password

1. Log in to your account
2. Navigate to Profile/Settings
3. Click "Change Password"
4. Enter current password
5. Enter new password (must be at least 8 characters with uppercase, lowercase, and number)
6. Confirm new password
7. Click "Update"

## User Roles

### Clinical Staff

**Doctors** have full access to:
- View all patient records
- Edit all patient fields
- Add clinical notes
- Upload medical documents
- Manage care team members
- Access emergency patient records

**Nurses** have limited access to:
- View assigned patient records
- Edit medications and treatment summary only
- Add clinical notes (own notes only)
- View documents
- See care team information

### Patients

Can:
- View their own medical record (read-only)
- View care team members
- Download their documents
- View clinical notes

Cannot:
- Edit their record
- Add notes or documents
- Access other patients' records

### Guardians

Can:
- View assigned patient's information
- Contact care team members
- Access emergency information

## Clinical Staff Guide

### Viewing Patient Records

#### Search for Patient

1. Click "Search Patient" in the dashboard
2. Enter patient RUT (format: XX.XXX.XXX-X)
3. Click "Search"
4. Select patient from results

#### Access Patient Record

Once a patient is selected:

1. **General Tab**: View patient demographics and diagnosis
2. **Notes Tab**: Read and add clinical notes
3. **Documents Tab**: View and upload medical documents
4. **Care Team Tab**: Manage care team members

### Editing Patient Information

#### Edit Medications

1. Go to patient record
2. Click "Edit" on Medications card
3. Modify medications list
4. Click "Save"

#### Edit Treatment Summary

1. Go to patient record
2. Click "Edit" on Treatment Status card
3. Update summary text
4. Click "Save"

#### Add Emergency Contact

1. Go to patient record
2. Click "Edit" on Emergency Contacts
3. Enter contact details:
   - Name
   - Relationship
   - Phone number
4. Click "Save"

### Managing Clinical Notes

#### Add Note

1. Go to Notes tab
2. Click "Nueva Nota"
3. Enter note content
4. Click "Guardar Nota"

#### Edit Note

1. Locate note in list
2. Click edit icon
3. Modify content
4. Click "Guardar Cambios"

#### Delete Note

1. Locate note in list
2. Click delete icon
3. Confirm deletion

### Uploading Documents

#### Upload Medical Document

1. Go to Documents tab
2. Click "Subir Nuevo Documento"
3. Fill in:
   - Title (required)
   - Document type (exam, surgery, chemotherapy, etc.)
   - File (image or PDF, max 10MB)
   - Description (optional)
4. Click "Guardar Documento"

#### Upload Oncology Committee Document

1. Go to Documents tab
2. In "Comité Oncológico" section
3. Click "Subir Archivo" or "Tomar Foto"
4. Enter title for document
5. Click "Guardar"
6. Repeat for multiple documents

### Managing Care Team

#### Add Care Team Member

1. Go to patient record
2. Click Care Team tab
3. Click "Agregar Miembro"
4. Search for healthcare provider
5. Select role (Oncologist, Surgeon, Nurse, etc.)
6. Click "Agregar"

#### Remove Care Team Member

1. Go to Care Team tab
2. Click delete icon next to member
3. Confirm removal

## Patient Guide

### Accessing Your Record

1. Log in to the platform
2. Click "Mi Ficha Médica" (My Medical Record)
3. View your complete medical information

### Understanding Your Record

#### General Information
- Personal demographics
- Cancer type and stage
- Current medications
- Allergies
- Emergency contacts

#### Medical History
- Surgical procedures and dates
- Treatment summary
- Current clinical status

#### Care Team
- Names and roles of your healthcare providers
- Contact information (if available)

#### Documents
- Medical examination results
- Surgical reports
- Treatment documentation

#### Clinical Notes
- Notes added by your care team
- Consultation summaries

### Downloading Documents

1. Go to Documents section
2. Click document you want
3. Click "Descargar" (Download)
4. File will download to your device

## Emergency Access

### For Clinical Staff During Emergencies

If emergency access to a patient record is required:

1. Click "Acceso de Emergencia" on home page
2. Enter patient RUT
3. Verify emergency access code (provided by hospital administration)
4. Record will display in read-only mode
5. Contact information and critical alerts will be highlighted

### Important Notes

- Emergency access is logged and audited
- Limited time access only
- Cannot modify records in emergency access mode
- Must provide justification if required

## Troubleshooting

### Login Issues

**Problem**: Cannot log in
- Verify email and password are correct
- Check CAPS LOCK is off
- Reset password if forgotten
- Check if account is active (contact administrator)

**Problem**: Session expired
- Log in again
- Use browser session if available
- Close and reopen browser if needed

### Viewing Documents

**Problem**: Cannot download document
- Check file size is under 10MB
- Verify document upload completed successfully
- Check browser download settings
- Try different file format

**Problem**: Image won't display
- Verify file is supported format (JPG, PNG, PDF)
- Check file is not corrupted
- Refresh page and try again

### Editing Information

**Problem**: Cannot edit field
- Verify user role has permission (doctors can edit all fields, nurses have limited editing)
- Ensure patient record is not locked
- Check for validation errors in form
- Try again or contact administrator

### Performance

**Problem**: Application is slow
- Check internet connection speed
- Close other browser tabs
- Clear browser cache
- Refresh the page
- Try accessing during off-peak hours

**Problem**: Images not loading
- Check internet connection
- Verify file upload completed
- Clear browser cache
- Contact administrator if issue persists
